# アルバトロス 3D組立シミュレーター

次世代足場（アルバトロス）を3D空間で「線を引く感覚」で組み立て、部材数量をリアルタイムに自動拾い出しするWebアプリです。

## 使い方

```bash
npm install
npm run dev   # http://localhost:3000
```

### 操作方法（マウスだけで完結）

| 操作 | 動作 |
|---|---|
| 地面をクリック | 足場の列を描き始める |
| マウスを動かす | スパン割り（1829優先）されたプレビューが追従 |
| クリック | 区間を確定して続行（直交方向へ動かすと角になる） |
| ダブルクリック / Esc / Enter | 列を完成 |
| 右ドラッグ / ホイール | 視点回転 / ズーム |
| 「選択・編集」モードで部材クリック | 列を選択（スパン変更・階段化・削除・枠幅変更） |

- 段数・枠幅・巾木などの設定変更は、配置済みの足場すべてに**即時反映**されます
- 数量表は配置と常に連動し、CSV（規格コード・単重付き）でダウンロードできます

## アーキテクチャ

「パラメータが真実、部材は導出」— ストアに保存するのは配置パラメータ（列の原点・スパン列・枠幅）と全体設定のみで、部材の数量・3D形状は常にそこから計算します。スパン変更時に踏板や手すりが追従するのは更新処理ではなく参照の再計算であり、追従漏れが構造的に起きません。

```
src/features/scaffold/
├── catalog/albatross.ts   # 部材規格マスタ（単重・規格コード・アンチ敷き並べ規則）
├── model/                 # 型定義とスナップ・スパン割り・支柱組合せの純関数
├── logic/                 # 配置 → 数量表（BOM）→ CSV（React非依存）
├── store/                 # Zustand ストア（配置パラメータ＋UI状態のみ）
├── three/                 # React Three Fiber 描画・配置操作
└── ui/                    # ツールバー・設定・選択編集・数量表パネル
```

部材リスト・数量算出ルールは [terunosuke/sub-alba](https://github.com/terunosuke/sub-alba) の拾い出し計算に準拠しています。

## 技術スタック

Next.js (App Router) / TypeScript / React Three Fiber / Zustand / Tailwind CSS

## 入力項目・拾い出しの対応範囲（sub-alba 準拠）

- 段数・最上段900mm・枠幅・妻側面数（0/1/2）
- 支柱構成の自動解決＋手動調整（高さ整合チェック付き）
- ジャッキベース（なし／SB20／SB40／混在＋過不足チェック）・タイコ・根がらみ・敷板
- アンチ設置段（全段／最下段以外／段指定）、巾木（両面／片面／なし＋段指定）
- 階段（3Dでスパンを階段化・設置段指定）
  - 1クリックで隣のスパンとペアの「2スパン1セット」になり、斜め階段が2段を一気に登る
  - （列にスパンが1つしかない場合のみ垂直型1スパン）
  - 階段直上のアンチは段ごとに千鳥（交互）で開口
  - 拡幅 = 階段セット＋両隣の5列が1219幅に拡がり、外2列に追加建地＋短手布材305/914→1219差替/アンチ24→50差替
- 壁つなぎ（KTS16〜100、設置段・スパンの全部/千鳥/指定）
- 層間養生ネット＋ブラケット、外周メッシュシート・妻側シート
- フリーメモ（CSV末尾に出力）
- 出力: CSV（部材名/数量/単重/重量＋総重量）、Excel（規格コード・数量の発注書式）
- サマリー: 総重量・総高さ・スパン計・支柱構成・輸送目安（ユニック/平車）・車両分割提案
